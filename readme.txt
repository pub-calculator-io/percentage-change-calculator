== Usage ==

Add the Percentage Change Calculator shortcode to your page, post or sidebar:

`[ci_percentage_change_calculator]`

Add the following code to your template where you would like the Percentage Change Calculator to appear:

`<?php display_ci_percentage_change_calculator(); ?>`
